# Symfony-skeleton-full


## Introduction

## Command
composer create-project -s dev alexperegrina/symfony-skeleton-full path/project

## Vagrant
### Run
```
vagrant up
```
### Stop
```
vagrant halt
```

## Server local Symfony
```
symfony server:start
```

## Generate the SSL keys
```
bin/console lexik:jwt:generate-keypair
```
